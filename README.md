# DownloadXiaMi
根据虾米的[专辑/歌曲]URL 免虾币下载歌曲到本地

### 效率
本地测试：[此时时刻 巡回演唱会](http://www.xiami.com/album/2100200848) 25+1 首歌曲，全部下载成功，共133M，10M电信带宽，耗时1分38秒。

##### 简单的数学：1.25M/S * 98S = 122.5M 
说明本程序的速度还是相当可以的（电信带宽竟然超过了平均？）。
